Este projeto foi desenvolvido como parte do desafio da DIO: Escrevendo as Classes de um Jogo. A proposta é aplicar conceitos de programação orientada a objetos em JavaScript e usando Node no terminal.

--- Objetivo
Criar uma classe Heroi com as seguintes propriedades:

- nome
- idade
- tipo de heroi

E um método atacar() que imprime: "O {tipo} {nome} atacou usando {ataque}"

O tipo de ataque varia conforme o tipo do herói.
